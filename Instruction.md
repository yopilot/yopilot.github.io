# Video Conference & Screen Share Application

## Overview
A cutting-edge, peer-to-peer video conferencing and screen sharing application built with HTML, CSS, and JavaScript. Features a modern, sleek interface with sophisticated animations and a refined color palette optimized for professional and personal use.

## Core Features

### Video Windows Layout
- **4-Window Display**: Your video, your screen share, partner's video, partner's screen share
- **Flexible Fullscreen Mode**: Expand any window with smooth transitions; remaining windows appear as resizable, draggable overlays in PIP
- **Horizontal Flip**: Auto-invert your video feed on your screen only for natural interaction

### Audio & Video Quality
- **Dual Audio Streams**: Capture microphone audio and system audio (screen share)
- **Auto Quality Scaling**: Dynamically adjusts bitrate and resolution based on network conditions
- **P2P Optimization**: Direct peer-to-peer connection for minimal latency

### User Experience
- **Unique Peer ID**: Auto-generated, shareable connection identifier
- **Responsive Design**: Seamless experience across devices with fluid animations
- **Advanced Motion Effects**: Smooth micro-interactions, spring animations, and fade transitions
- **Professional Theming**: AI-optimized dark and light themes with adaptive color schemes

### Advanced Features
- **Draggable Windows**: Move and resize PIP overlays with momentum-based animations
- **Quality Indicators**: Real-time connection and stream quality display with animated visualizations
- **Smart Theme Detection**: Auto-switching dark/light modes with preference persistence

## Technical Stack
- **P2P Framework**: PeerJS
- **Audio/Video**: WebRTC
- **Screen Capture**: Screen Capture API
- **Frontend**: Vanilla JavaScript, CSS Grid/Flexbox, modern CSS animations

## Best Practices
- Modular, maintainable code architecture
- Progressive enhancement for legacy browsers
- Accessibility compliance (WCAG 2.1)
- Performance optimized for low-bandwidth scenarios
